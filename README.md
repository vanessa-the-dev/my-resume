# Vanessa Nellyn Isidro — Interactive Résumé 🔎

A playful, single-page résumé built with Vite + React and Tailwind CSS that looks and behaves like a Google search engine.

---

## ✨ Features

- **Search-style landing page** — a "Vanessa" wordmark over a Google-like search box.
- **Autocomplete suggestions** — start typing and a dropdown filters résumé prompts in real time, bolding the part you haven't typed yet.
- **Full keyboard navigation** — Arrow Up/Down to move through suggestions, Enter to search the highlighted one, Esc to dismiss.
- **Mouse support** — hover to highlight and click to search; the dropdown closes on any outside click.

---

## 🚀 Getting started

No install required. Just open the file:

```bash
# from the project folder
start index.html      # Windows
# or simply double-click index.html in your file explorer
```

Optionally, serve it locally (any static server works):

```bash
npx serve .
# then visit the printed http://localhost:... URL
```
